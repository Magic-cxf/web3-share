### 1.引言

#### 1.1 Web 3的概念

- 互联网的发展历程：Web 1.0→
- Web 3的核心理念：去中心化、用户数据自主权、无信任机制

#### 1.2 为什么需要Web 3?

- 中心化系统的弊端：数据垄断、隐私问题、单点故障
- Web 3 是如何解决这些问题的：去中心化的优势与挑战
